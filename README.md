# everyplanet.github.io
## 2022 Computer Science ALT 1 Project : Concerts In Town
>Note: This project was used for practicing Firebase/Javascript and is not related to the official Leaving Cert 2022 exam.

## About
Concerts In town is a basic concert-listings site inspired by [BandsInTown](https://www.bandsintown.com/) and [TicketMaster](https://www.ticketmaster.ie/).
Using an Artist creation and Concert listing form seen on the **New Submissions** page, users/artists can create accounts with their details that can be viewed on the **Artists** page, and concert listings which can be viewed on the **Concerts** page.

## Resources Used
Much of the website uses [Bootstrap](https://getbootstrap.com/) for its more complex and stylistic elements.
For storage, a [Firebase](https://firebase.google.com/) realtime database is connected to each page which collects or displays data.
